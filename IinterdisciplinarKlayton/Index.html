Exercícios Escritos





"Esse programa pede ao usuário para inserir um número, verifica se ele é positivo, negativo ou zero, e exibe uma mensagem correspondente."
-> Ir para o primeiro exercício



"Esse programa pede ao usuário para inserir sua idade e, com base nela, exibe uma mensagem informando se ele é criança (0-12), adolescente (13-17), adulto (18-64) ou idoso (65 anos ou mais) e também contém segredos que começe os jogos!."
-> Ir para o segundo exercício



"Esse programa pede ao usuário para inserir um número correspondente a um dia da semana (1 para Domingo, 2 para Segunda, etc.) e utiliza a estrutura switch para exibir o nome do dia correspondente."
-> Ir para o terceiro exercício



"O programa imprime os números de 1 a 10 utilizando um laço For."
-> Ir para o quarto exercício



"Este programa solicita que o usuário insira números inteiros repetidamente até que o número zero seja digitado. Após a entrada do número zero, o programa calculará e exibirá a soma de todos os números que foram inseridos, excluindo o zero."
-> Ir para o quinto exercício



"Este programa solicita que o usuário tente adivinhar um número entre 1 e 10. O programa continuará pedindo ao usuário para fazer novas tentativas até que o número correto seja adivinhado. Quando o usuário finalmente acertar o número, o programa exibirá uma mensagem de parabéns."
-> Ir para o sexto exercício



"Crie um programa que peça ao usuário para inserir 5 números inteiros e os armazene em um vetor. Em seguida, exiba o maior e o menor inserido."
-> Ir para o sétimo exercício



"Este programa inclui uma função chamada encontrarMedia, que recebe um vetor de números como argumento e retorna a média desses números. O programa pede ao usuário para informar a quantidade de números que deseja fornecer, armazena esses números em um vetor e, em seguida, utiliza a função encontrarMedia para calcular e exibir a média dos números inseridos."
-> Ir para o oitavo exercício



"Este programa percorre um vetor de números e armazena em um novo vetor apenas os números que são maiores que um valor específico fornecido. Após filtrar os números, o programa exibe o novo vetor contendo apenas os números que atendem a essa condição."
-> Ir para o nono exercício



"Este programa percorre um vetor de palavras e conta quantas vezes uma palavra específica aparece nesse vetor. O resultado da contagem é armazenado em uma variável chamada contagem, e o programa exibe esse valor."
Ir para o décimo exercício
